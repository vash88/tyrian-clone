# Rewrite Prototype

A separate web-native prototype that recreates the Tyrian-style combat and shop loop with original placeholder visuals and TypeScript + Canvas.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Prototype scope

- one authored vertical stage
- keyboard-only combat
- front and rear weapons with power upgrades
- shield and generator upgrades
- left and right sidekick slots
- stage clear to shop to next sortie loop
